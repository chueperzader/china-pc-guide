# china-pc-guide

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面主要围绕特定主题提供参考信息，不针对任何单个域名或具体网站。

## 项目简介

本仓库旨在以静态 HTML 页面的形式，整理和展示一系列与“电脑使用指南”相关的资料。每个页面都是独立的，便于浏览和分享。这些页面内容可能涵盖工具使用、配置参考、常见问题解答等，但不限于特定平台或服务。

## 目录说明

- `docs/`：存放所有 HTML 页面的主目录。每个文件为一个独立的归档页面。
- `assets/`：存放页面所需的静态资源，如图片、样式表或脚本文件（如有）。
- `README.md`：本文件，提供仓库的基本说明。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，可直接在浏览器中打开查看。
- 页面内容均为静态文本和样式，无需后端服务支持。
- 页面标题和内容均围绕通用电脑使用指导展开，不指向任何具体商业网站或服务。

## 维护说明

- 欢迎通过 Issues 提交建议或报告问题，但请不要提交与具体域名或网站推广相关的内容。
- 本仓库不接收包含商业推广、诱导点击或违反平台规定的 PR。
- 页面内容更新将不定期进行，以保持信息的时效性和准确性。

## 使用方式

1. 克隆或下载本仓库。
2. 进入 `docs/` 目录，用浏览器打开任意 `.html` 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
